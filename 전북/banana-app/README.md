# 바나나배관 올케어 (전북 전주점)

[![Netlify Status](https://api.netlify.com/api/v1/badges/242da41a-ae25-4192-a2ae-550299d4981e/deploy-status)](https://app.netlify.com/projects/bananajeonju/deploys)

이 프로젝트는 전북 전주 지역 바나나배관 올케어를 위한 Next.js 프로젝트입니다.
SEO를 위해 지역+키워드 조합에 해당하는 숫자 ID 기반의 정적 페이지를 생성합니다.

## 도메인
https://bananajeonju.netlify.app

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

이 명령어는 `out` 디렉토리에 정적 내보내기 파일을 생성하고 올바른 도메인으로 `sitemap.xml`을 생성합니다.
